# Tournoi sportif à élimination directe (tournoi_sportif)

## Contexte
Un tournoi à élimination directe se modélise naturellement par un arbre binaire.

## Objectifs pédagogiques
- Arbres binaires complets
- Récursivité
- Parcours par niveau

## Travail demandé
- Initialisation du tournoi
- Enregistrement des scores
- Affichage du tableau
- Détermination du champion
- Calcul du nombre de tours
- Recherche du parcours d’un joueur
